第四章巩固练习题

1. 键盘输入身高体重，计算BMI （20分）

分类 BMI 范围
偏瘦 <= 18.4
正常 18.5 ~ 23.9
过重 24.0 ~ 27.9
肥胖 >= 28.0

目标 ： 根据 BMI的值 输出 对应的分类

提示 ： 体质指数(BMI)=体重(kg)÷身高(m)^2

2. 判断两个数值的大小关系 （15分）

目标：从键盘输入两个数字，判断其大小关系，并输出结果

3. 判断输入的字母是大写还是小写字母，如果是大写则输出其对应的

小写形式，反之输出大写形式。（20分）

4.用switch语句打印数字对应的星期数；（15分）

数字1-7 分别对应星期一到星期天

5.用switch语句来计算一元二次方程的解 （30分）

Example
Input
Input a: 4
Input b: -2
Input c: -10

Output

Root1: 1.85
Root2: -1.35
